Example — image prediction (Python)
import requests url = "http://localhost:8000/predict/image" files = {"file": open("leaf.jpg", "rb")} res = requests.post(url, files=files).json() print(res["prediction"]["class"]) print(res["prediction"]["confidence"]) print(res["result_url"])
Example — video prediction (Python)
import requests url = "http://localhost:8000/predict/video" files = {"file": open("potato_video.mp4", "rb")} res = requests.post(url, files=files).json() print(res["dominant_class"]) print(res["class_distribution"]) print(res["result_url"])
Example — curl
curl -X POST http://localhost:8000/predict/image \ -F "file=@leaf.jpg" | python3 -m json.tool
